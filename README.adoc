= Скрипт для соединения видео, аудио и субтитров в единый .mkv файл

== Описание

Многие сериалы распространяются с субтитрами (и даже порой аудио) в отдельных файлах. Так как не все проигрыватели поддерживают чтение субтитры из той же папки,
мной был создан скрипт, который вшивает субтитры в mkv и конвертирует видео кодеком hevc, что позволяет уменьшить размер видео
